# Aufgabe 1

Diese Übung dreht sich darum, den Umgang mit Tailwind im Kontext einer React-App zu üben.

Deine Aufgabe ist es, dieses Design mit Tailwind CSS nachzubauen: https://www.figma.com/community/file/1342500275468274237/cart-page

Beginne damit, eine neue React-App in diesem Verzeichnis zu erstellen und sie für die Verwendung von Tailwind einzurichten. Versuche dann, so viele Seiten des Referenzdesigns wie möglich zu erstellen. Erstelle eine `App` Komponente, die jeweils eine Seite nach der anderen rendert. Du kannst die Seiten, an denen du nicht arbeitest, auskommentieren:

```jsx
function App() {
    return (
        <div className="App">
            <DeliveryDetailsPage />
            {/* <CartPage /> */}
            {/* <PaymentPage /> */}
        </div>
    );
}
```

Dies ist absichtlich eine sehr große und schwierige Übung, also mach dir keine Sorgen, wenn du nicht alles schaffst. Das Ziel ist es, dich mit Tailwind CSS vertraut zu machen und gleichzeitig zu üben, dich lange auf ein einzelnes Problem zu konzentrieren. Arbeite einfach weiter und komm so weit wie möglich. Verwende keine vordefinierten Komponenten oder Bibliotheken, sondern nur Tailwind CSS Klassen selbst. Denke an die verschiedenen Ressourcen, die dir zur Verfügung stehen, falls du feststeckst.
